# Insight works Demo

## 项目介绍

智造分析平台 Demo 是一个基于 Vue 3 + .NET Core 的智能制造数据分析系统。前端基于 [vue-pure-admin](https://github.com/pure-admin/vue-pure-admin) 模板开发,后端使用 .NET Core Web API。

## 功能特性

- 厂商管理
  - 厂商信息管理(新增/编辑/删除)
- 设备管理
  - 设备基本信息管理(新增/编辑/删除)
  - 设备数据同步
  - 同步日志查看
- 产品型号管理
  - 型号信息管理(新增/编辑/删除)
- 统计分析
  - 机况记录表
  - 生产记录表
  - 机况甘特图
  - 生产平均时间图
